zip2city python script
======================
anthony.dandrea@me.com

1st argument is an excel file with .csv file extension.
2nd argument is optional. If supplied it will specify the output file name. No extension necessary. If not supplied it will use the first argument name with '_output' affixed to the name.
Example usage: python zip2city.py myzipcodesheet.csv
Example output: myzipcodesheet_output.csv
